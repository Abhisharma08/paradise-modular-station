# **App Name**: OfficeEase Landing

## Core Features:

- Lead Capture Form: Multi-step lead capture form on the top right to collect user information.
- Data Storage: Data persistence to Google Sheets after each step of the lead capture form.
- USP Display: Display USPs as bullet points to highlight key benefits.
- Furniture Showcase: Display images of furniture to build trust and showcase products.
- Final CTA: Clear call-to-action at the end of the page to encourage user engagement.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust.
- Background color: Light gray (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Orange (#FF9800) to highlight calls to action and important elements, such as buttons.
- Body and headline font: 'Inter', a sans-serif font for clear readability.
- Note: currently only Google Fonts are supported.
- Use modern, line-style icons to represent USPs and key features.
- Ensure a clean, responsive layout that adapts to different screen sizes.
- Subtle fade-in animations on scroll to enhance user engagement without distraction.
